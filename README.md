# 📌 Overview

LifeBridge AI is a web-based disaster management platform designed to help citizens, volunteers, hospitals, and emergency response teams during natural disasters and emergencies.

The application provides real-time disaster monitoring, interactive maps, SOS management, AI assistance, community support, and disaster simulation in one platform.
